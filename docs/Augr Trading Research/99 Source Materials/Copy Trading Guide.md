# COPY TRADING GUIDE FOR POLYMARKET

(full video + exact settings included)

I spent months testing different tools, setups and configs.

Lost everything at the start.

Then made it back and started earning once the setup was right.

This cost me a lot of time, you need ~3 minutes:

1. Open this bot: https://t.me/KreoPolyBot?start=ref-start…

I tried different tools, but this one has more advanced features and the fastest execution.

2. Click Copy trade -> Add Task -> W1

3. Paste wallet address and name it

Use particular names like "AI trader", "sports 85% win rate whale" etc.

4. After entering the amount choose the exact setup:
	1. Fixed (same trading amount every time)
	2. Exact (each trade is the same as main trader's)
	3. Portfolio based (proportional to your balance)

You have to use the one that fits trader's beaviour.

How to choose?

- Use Fixed for traders with < 5 trades per day

- Exact when trader’s portfolio size is similar to mine

- Portfolio for traders who open many small positions

5. Change Slippage to 5%, the default 10% is too much

6. Set Min Trigger to $100 to avoid mirroring small transactions

7. Set Min Liquidity to $20,000 to avoid volatile markets

8. Turn on Copy Sells to not just buy, but sell automatically as well

9. Set Stop Loss to 70%, this feature will manage your risk

10. Click Back -> Start

Congrats, now your bot runs automatically.
